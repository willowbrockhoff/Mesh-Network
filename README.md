This code involves firmware (/NodeCode)that is flashed onto an ESP32, as well as a flutter application (/App/mesh). 
The flutter application can be built after following these download steps: https://docs.flutter.dev/install
*The  Flutter app may be ran on an Andriod emulator but emulators lack BLE capabilities so the app will not be able to scan/add nodes*
The firmware can be built with installation of PlatformIO (https://platformio.org/install), but without a node to flash to it is pointless.
